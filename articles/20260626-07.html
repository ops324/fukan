Samsung、SmartThings API を 10月に有料化へ

Samsung が SmartThings API へのアクセスを 10月から有料化。月額4.99ドルの個人開発者向けプランから法人プランまで段階的に導入。

俯瞰 編集部 — AI 自動要約 + 人手編集
𝕏 B!

Samsung は SmartThings API について、2026年10月より有料化を実施すると発表した。これまで API は無料でアクセス可能だったが、同社は『より高品質なサービス提供』を理由に複数の価格帯を設定する。

個人開発者向けには月額4.99ドルの『Personal Plan』が用意される。一方、商用・大規模利用向けにはより高額なプランが用意される見通しだ。Samsung は詳細な料金体系と移行スケジュールをまもなく発表予定としている。

この変更は、Home Assistant など SmartThings API を統合したサードパーティーのスマートホームプラットフォームに影響を与える可能性が高い。Home Assistant は「今回の変更により、その統合は『Personal Plans』の対象になる」と声明を出している。

スマートホーム愛好家で API に直接アクセスする高度なユーザーも影響を受ける。これまで無料でカスタマイズ可能な自動化を構築できていたが、今後は月額利用料が発生することになる。

Samsung は開発者パートナーシップメール（partners@smartthings.com）で詳細情報提供予定。業界では、API 有料化がスマートホーム市場の競争環境に与える影響について議論が高まっている。
