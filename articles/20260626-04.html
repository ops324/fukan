AWS、OpenAI モデル GPT-5.5・Codex の提供開始

OpenAI と AWS の戦略的パートナーシップが拡大。GPT-5.5・GPT-5.4・Codex が Amazon Bedrock で一般提供開始。Azure 独占時代が終焉。

俯瞰 編集部 — AI 自動要約 + 人手編集
𝕏 B!

Amazon Web Services（AWS）は、OpenAI の最新モデルを Amazon Bedrock 経由で提供開始した。これまで OpenAI モデルは Microsoft Azure 経由でのみ入手可能だったが、AWS ユーザーも直接アクセスできるようになった。

Bedrock 上で提供されるのは、OpenAI の最先端モデルである GPT-5.5 と GPT-5.4、そしてコーディングエージェント「Codex」である。両モデルは Bedrock の次世代推論エンジン上で動作し、高いパフォーマンス、信頼性、セキュリティが実現される。

このパートナーシップ拡大は、OpenAI と Microsoft が4月末に契約条件を修正したことで可能になった。修正により、OpenAI は Azure 以外のクラウドプロバイダーでもモデルを配信する自由を得た。料金は OpenAI の公式料金と同じで追加費用は発生しない。

Codex は Codex App、CLI、IDE 統合を通じて利用でき、すべての推論は Amazon Bedrock 経由で実行される。IAM、VPC 隔離、暗号化といった AWS セキュリティ機能がフルに適用される。この拡大により、AWS ユーザーは OpenAI の最先端モデルに直接アクセスでき、Azure から乗り換えるインセンティブが生まれた。
